#+TITLE:Memory Layout

The memory layout is setup like this:

| *Name*    | RAM   | Tape interface | ROM   |
|-----------+-------+----------------+-------|
| *Address* | $0000 | $F7FF          | $F800 |
|-----------+-------+----------------+-------|
| *Size*    | $F7FF | $0001          | $0800 |
|-----------+-------+----------------+-------|
| *r/w*     | r/w   | r/w            | r     |
